## 资源使用指南

**本地资源优先**: 前端实现时必须优先使用 `Resources/` 目录中提供的资源文件，禁止用外部 URL 替代本地应有的资源。

- `Resources/icons/` — SVG 图标文件，直接引用
- `Resources/tokens.css` — CSS 变量，导入项目样式
- `Resources/tailwind.config.js` — Tailwind 扩展配置，合并到项目配置中
- 如有资源缺失，请在 log.md 中记录并通知 Tech Lead，不要自行用外部 URL 替代
